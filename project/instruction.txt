ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОГРАММЫ

1) для отрисовки графических данных используется Jupyter Notebook.
В случае отсутствия необходимых python-библиотек для начала необходимо запустить файл ./install.sh,
(который установит python-pip и runipy, а также две библиотеки numpy, matplotlib).
(Для этого может потребоваться право на исполнение данного .sh файла, которое можно получить командой chmod +x *.sh);
2) далее файлы проекта компилируются командой ./Make.sh;
3) затем выполняется команда запуска ./run.sh непостредственно программы (при необходимости, нужное число исполнителей выставляется в файле run.sh);
4) после отработки jupyter notebook сохранит полученные данные в виде pdf файла в этой же директории.
